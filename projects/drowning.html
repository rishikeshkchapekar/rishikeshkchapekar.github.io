Drown Alarm

This is a security software designed to raise an alarm when it detects someone drowning

About Drown Alarm

About Drown Alarm

Drown alarm uses the live feed of a security camera watching over a swimming pool

Convolutional Neural Networks are used to analyse a sequence of frames, detect a person in it, and determine whether he/she is drowning

How It Works

This security system took input from a swimming pool's security camera's live feed, and analysed a sequence of frames to determine if there was someone drowning in it.

The drown detection was done using Convolutional Neural Networks, which were trained on a large dataset of multiple frames of people drowning against frames of people not drowning

It was necessary to consider a sequence of images rather than a single one as there would be cases where the swimmer was taking a dive or simply swimming underwater. This system was able to differentiate between such cases most of the time
