About GesControl

GesControl is a purely electronics based project which interfaced with a python client code on the target laptop

Input was taken on an Arduino through a few sensors, processed,and sent to the client code for executing various functions

An AI based version is under development, which uses image processing to interpret the gestures and execute functions

Working

Gesture Recognition:

A wide array of IR sensors was used, each of which recorded a certain input upon being triggered, which collectively formed a pattern

Understanding Commands:

The pattern was sent to the client running a Python code. The client code communicated directly with the host system and controlled the system's functions

Functions:

Basic Mouse Functions: Two ultrasonic sensors were used to note the movement of the user's hands and accordingly move the mouse cursor left or right. A threshold distance was set,beyond which the action would be considered for a mouse click

Keyboard Shortcuts: A lot of the commonly used keyboard shorcuts were assigned to certain sensor reading patterns

Function Keys: Commonly used function keys such as those for volume control and screen brightness were assigned to certain patterns

Switch Between Workspaces: Since it was built on a Linux platform,a "swipe" option was also assigned for switching between workspaces on the screen

Keyboard And Typing: A Python virtual keyboard was built which would pop after making a certain gesture. Localising a group of keys based on the position of the user's hand brought the cursor over a small set of keys, and then other gestures were used to move the cursor around and click on the keys

Power Off: A gesture was assigned, which would make the client code kill all running processes on the system, and shutdown the system
